# Licença

O VMark é lançado sob a Licença ISC — uma licença simples e permissiva semelhante à MIT. Veja o que isso significa para você.

O VMark é lançado sob a **Licença ISC** — uma das licenças de código aberto mais simples e permissivas.

## O que a Licença ISC Permite

- **Usar** — para qualquer finalidade, comercial ou pessoal
- **Copiar** — distribuir livremente
- **Modificar** — alterar o código como desejar
- **Distribuir** — compartilhar suas versões modificadas

## O que ela Exige

- Manter o aviso de direitos autorais e o aviso de permissão nas cópias

## O que ela Não Faz

- Fornecer qualquer garantia — o software é fornecido "como está"
- Responsabilizar o autor por danos

## Por que ISC?

A Licença ISC é funcionalmente equivalente à Licença MIT, mas com linguagem mais simples. É aprovada pela [Open Source Initiative (OSI)](https://opensource.org/licenses/ISC) e é a licença padrão usada pelo npm e pelo OpenBSD.

Foi escolhida para o VMark porque:

- **Simplicidade** — fácil de ler e entender
- **Permissividade** — sem restrições sobre como você usa o software
- **Compatibilidade** — funciona com praticamente todas as outras licenças de código aberto
