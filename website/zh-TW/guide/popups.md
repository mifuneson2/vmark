# 行內彈出視窗

VMark 提供情境式彈出視窗，用於編輯連結、圖片、媒體、數學、腳注等內容。這些彈出視窗在所見即所得和原始碼兩種模式下均可使用，並支援一致的鍵盤導覽。

## 通用鍵盤快捷鍵

所有彈出視窗共用以下鍵盤行為：

| 操作 | 快捷鍵 |
|------|--------|
| 關閉/取消 | `Escape` |
| 確認/儲存 | `Enter` |
| 在欄位間導覽 | `Tab` / `Shift + Tab` |

## 連結提示與彈出視窗

VMark 對連結採用兩層系統：懸停時顯示唯讀提示，透過快捷鍵開啟編輯彈出視窗。

### 懸停提示（唯讀）

**觸發條件：** 懸停在連結上（300ms 延遲）

**顯示內容：**
- **URL 預覽** — 截短的 URL，懸停時顯示完整 URL
- **開啟按鈕** — 在瀏覽器中開啟連結（或跳至 `#書籤` 對應的標題）

**行為：** 唯讀。移開滑鼠即關閉。

### 編輯現有連結

**觸發條件：** 將游標置於連結中 + `Mod + K`

**欄位：**
- **URL** — 編輯連結目標
- **開啟** — 在瀏覽器中開啟連結
- **複製** — 複製 URL 至剪貼簿
- **刪除** — 移除連結，保留文字

### 建立新連結

**觸發條件：** 選取文字 + `Mod + K`

**智慧剪貼簿：** 若剪貼簿中有 URL，會自動填入。

**欄位：**
- **URL 輸入** — 輸入目標位址
- **確認** — 按 Enter 或點選 ✓
- **取消** — 按 Escape 或點選 ✗

### 原始碼模式

- **`Cmd + 點擊`** 連結 → 在瀏覽器中開啟
- **點擊** `[text](url)` 語法 → 顯示編輯彈出視窗
- 在連結內按 **`Mod + K`** → 顯示編輯彈出視窗

::: tip 書籤連結
以 `#` 開頭的連結被視為書籤（內部標題連結）。點選「開啟」會跳至對應標題，而非開啟瀏覽器。
:::

## 媒體彈出視窗（圖片、影片、音訊）

統一的彈出視窗，用於編輯所有媒體類型 — 圖片、影片和音訊。

### 編輯彈出視窗

**觸發條件：** 雙擊任何媒體元素（圖片、影片或音訊）

**通用欄位（所有媒體類型）：**
- **來源** — 檔案路徑或 URL

**依類型分類的欄位：**

| 欄位 | 圖片 | 影片 | 音訊 |
|------|------|------|------|
| 替代文字 | 是 | — | — |
| 標題 | — | 是 | 是 |
| 封面圖 | — | 是 | — |
| 尺寸 | 唯讀 | — | — |
| 行內/區塊切換 | 是 | — | — |

**按鈕：**
- **瀏覽** — 從檔案系統選取檔案
- **複製** — 複製來源路徑至剪貼簿
- **刪除** — 移除媒體元素

**快捷鍵：**
- `Mod + Shift + I` — 插入新圖片
- `Enter` — 儲存變更
- `Escape` — 關閉彈出視窗

### 原始碼模式

在原始碼模式中，點擊圖片語法 `![alt](path)` 會開啟相同的媒體彈出視窗。媒體檔案（影片/音訊副檔名）在懸停時會顯示帶有原生播放控制項的浮動預覽。

## 數學彈出視窗

編輯 LaTeX 數學表達式，並提供即時預覽。

**觸發條件：**
- **所見即所得：** 點擊行內數學 `$...$`

**欄位：**
- **LaTeX 輸入** — 編輯數學表達式
- **預覽** — 即時渲染預覽
- **錯誤顯示** — 顯示 LaTeX 錯誤與實用語法提示

**快捷鍵：**
- `Mod + Enter` — 儲存並關閉
- `Escape` — 取消並關閉
- `Shift + Backspace` — 刪除行內數學（即使有內容也有效）
- `Alt + Mod + M` — 插入新行內數學

::: tip 錯誤提示
若有 LaTeX 語法錯誤，彈出視窗會顯示實用建議，如缺少大括號、未知指令或括號不對稱等。
:::

::: info 原始碼模式
在原始碼模式中，直接在文字中編輯數學內容。預覽顯示在 Mermaid/數學預覽面板中。
:::

## 腳注彈出視窗

行內編輯腳注內容。

**觸發條件：**
- **所見即所得：** 懸停在腳注參考 `[^1]` 上

**欄位：**
- **內容** — 多行腳注文字（自動調整高度）
- **跳至定義** — 跳至腳注定義位置
- **刪除** — 移除腳注

**行為：**
- 新腳注會自動聚焦內容欄位
- 文字區域隨輸入自動擴展

## Wiki 連結彈出視窗

編輯 Wiki 風格連結，用於文件內部連結。

**觸發條件：**
- **所見即所得：** 懸停在 `[[target]]` 上（300ms 延遲）
- **原始碼：** 點擊 Wiki 連結語法

**欄位：**
- **目標** — 工作區相對路徑（`.md` 副檔名自動處理）
- **瀏覽** — 從工作區選取檔案
- **開啟** — 開啟連結的文件
- **複製** — 複製目標路徑
- **刪除** — 移除 Wiki 連結

## 表格情境選單

快速表格編輯操作。

**觸發條件：**
- **所見即所得：** 使用工具列或鍵盤快捷鍵
- **原始碼：** 右鍵點擊表格儲存格

**操作：**
| 操作 | 說明 |
|------|------|
| 在上方/下方插入列 | 在游標處新增列 |
| 在左側/右側插入欄 | 在游標處新增欄 |
| 刪除列 | 移除目前列 |
| 刪除欄 | 移除目前欄 |
| 刪除表格 | 移除整個表格 |
| 欄靠左/置中/靠右對齊 | 設定目前欄的對齊方式 |
| 全部靠左/置中/靠右對齊 | 設定所有欄的對齊方式 |
| 格式化表格 | 自動對齊表格欄位（美化 Markdown） |

## 拼字檢查彈出視窗

透過建議修正拼字錯誤。

**觸發條件：**
- 右鍵點擊拼字錯誤的單詞（紅色底線）

**操作：**
- **建議** — 點擊以替換為建議內容
- **加入字典** — 停止將其標記為拼字錯誤

## 模式比較

| 元素 | 所見即所得編輯 | 原始碼 |
|------|--------------|--------|
| 連結 | 懸停提示 / `Mod+K` | 點擊 / `Mod+K` / `Cmd+點擊` 開啟 |
| 圖片 | 雙擊 | 點擊 `![](path)` |
| 影片 | 雙擊 | — |
| 音訊 | 雙擊 | — |
| 數學 | 點擊 | 直接編輯 |
| 腳注 | 懸停 | 直接編輯 |
| Wiki 連結 | 懸停 | 點擊 |
| 表格 | 工具列 | 右鍵選單 |
| 拼字檢查 | 右鍵 | 右鍵 |

## 彈出視窗導覽技巧

### 焦點流程
1. 彈出視窗開啟時，第一個輸入欄位自動獲得焦點
2. `Tab` 向前移動至各欄位和按鈕
3. `Shift + Tab` 向後移動
4. 焦點在彈出視窗內循環

### 快速編輯
- 簡單修改 URL：編輯後按 `Enter`
- 取消：從任意欄位按 `Escape`
- 多行內容（腳注、數學）：使用 `Mod + Enter` 儲存

### 滑鼠行為
- 點擊彈出視窗外部以關閉（變更被捨棄）
- 懸停彈出視窗（連結、腳注、Wiki）在顯示前有 300ms 延遲
- 將滑鼠移回彈出視窗可保持開啟
