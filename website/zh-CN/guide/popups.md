# 内联弹窗

VMark 为编辑链接、图片、媒体、数学公式、脚注等内容提供上下文弹窗。这些弹窗在所见即所得模式和源码模式中均可使用，并支持一致的键盘导航。

## 通用键盘快捷键

所有弹窗共享以下键盘行为：

| 操作 | 快捷键 |
|------|--------|
| 关闭/取消 | `Escape` |
| 确认/保存 | `Enter` |
| 在字段间导航 | `Tab` / `Shift + Tab` |

## 链接工具提示与弹窗

VMark 为链接提供两层系统：悬停时显示只读工具提示，通过键盘快捷键打开编辑弹窗。

### 悬停工具提示（只读）

**触发方式：**悬停在链接上（300ms 延迟）

**显示内容：**
- **URL 预览**——截断显示的 URL，悬停时显示完整 URL
- **打开按钮**——在浏览器中打开链接（对于 `#书签` 链接则跳转到对应标题）

**行为：**仅供查看。将鼠标移走即可关闭。

### 编辑已有链接

**触发方式：**将光标放在链接内 + `Mod + K`

**字段：**
- **URL**——编辑链接目标
- **打开**——在浏览器中打开链接
- **复制**——将 URL 复制到剪贴板
- **删除**——移除链接，保留文本

### 创建新链接

**触发方式：**选中文本 + `Mod + K`

**智能剪贴板：**如果剪贴板中有 URL，会自动填入。

**字段：**
- **URL 输入**——输入目标地址
- **确认**——按 Enter 或点击 ✓
- **取消**——按 Escape 或点击 ✗

### 源码模式

- **`Cmd + 点击`**链接 → 在浏览器中打开
- **点击** `[文本](url)` 语法 → 显示编辑弹窗
- 在链接内按 **`Mod + K`** → 显示编辑弹窗

::: tip 书签链接
以 `#` 开头的链接会被视为书签（内部标题链接）。打开时跳转到对应标题，而不是打开浏览器。
:::

## 媒体弹窗（图片、视频、音频）

用于编辑所有媒体类型——图片、视频和音频——的统一弹窗。

### 编辑弹窗

**触发方式：**双击任意媒体元素（图片、视频或音频）

**所有媒体类型的通用字段：**
- **来源**——文件路径或 URL

**类型专属字段：**

| 字段 | 图片 | 视频 | 音频 |
|------|------|------|------|
| 替代文本 | 是 | — | — |
| 标题 | — | 是 | 是 |
| 封面图 | — | 是 | — |
| 尺寸 | 只读 | — | — |
| 内联/块级切换 | 是 | — | — |

**按钮：**
- **浏览**——从文件系统选择文件
- **复制**——将来源路径复制到剪贴板
- **删除**——移除媒体元素

**快捷键：**
- `Mod + Shift + I`——插入新图片
- `Enter`——保存更改
- `Escape`——关闭弹窗

### 源码模式

在源码模式中，点击图片语法 `![alt](path)` 会打开相同的媒体弹窗。媒体文件（视频/音频扩展名）在悬停时会显示带有原生播放控件的浮动预览。

## 数学公式弹窗

带有实时预览的 LaTeX 数学表达式编辑器。

**触发方式：**
- **所见即所得：**点击内联数学公式 `$...$`

**字段：**
- **LaTeX 输入**——编辑数学表达式
- **预览**——实时渲染预览
- **错误显示**——显示 LaTeX 错误及有用的语法提示

**快捷键：**
- `Mod + Enter`——保存并关闭
- `Escape`——取消并关闭
- `Shift + Backspace`——删除内联数学公式（即使非空也有效）
- `Alt + Mod + M`——插入新内联数学公式

::: tip 错误提示
当出现 LaTeX 语法错误时，弹窗会显示有用的建议，例如缺少花括号、未知命令或括号不匹配。
:::

::: info 源码模式
在源码模式中，直接在文本中编辑数学公式。预览会显示在 Mermaid/数学预览面板中。
:::

## 脚注弹窗

内联编辑脚注内容。

**触发方式：**
- **所见即所得：**悬停在脚注引用 `[^1]` 上

**字段：**
- **内容**——多行脚注文本（自动调整高度）
- **跳转到定义**——跳转到脚注定义处
- **删除**——移除脚注

**行为：**
- 新脚注自动聚焦到内容字段
- 文本区域随输入自动扩展

## Wiki 链接弹窗

编辑用于内部文档连接的 Wiki 风格链接。

**触发方式：**
- **所见即所得：**悬停在 `[[目标]]` 上（300ms 延迟）
- **源码：**点击 wiki 链接语法

**字段：**
- **目标**——工作区相对路径（自动处理 `.md` 扩展名）
- **浏览**——从工作区选择文件
- **打开**——打开链接文档
- **复制**——复制目标路径
- **删除**——移除 wiki 链接

## 表格上下文菜单

快速表格编辑操作。

**触发方式：**
- **所见即所得：**使用工具栏或键盘快捷键
- **源码：**右键点击表格单元格

**操作：**
| 操作 | 描述 |
|------|------|
| 在上方/下方插入行 | 在光标处添加行 |
| 在左侧/右侧插入列 | 在光标处添加列 |
| 删除行 | 移除当前行 |
| 删除列 | 移除当前列 |
| 删除表格 | 移除整个表格 |
| 列左对齐/居中/右对齐 | 设置当前列的对齐方式 |
| 全部左对齐/居中/右对齐 | 设置所有列的对齐方式 |
| 格式化表格 | 自动对齐表格列（美化 Markdown） |

## 拼写检查弹窗

通过建议修复拼写错误。

**触发方式：**
- 右键点击拼写错误的单词（红色下划线）

**操作：**
- **建议**——点击以替换为建议内容
- **添加到词典**——停止将其标记为拼写错误

## 模式对比

| 元素 | 所见即所得编辑 | 源码 |
|------|--------------|------|
| 链接 | 悬停工具提示 / `Mod+K` | 点击 / `Mod+K` / `Cmd+点击`打开 |
| 图片 | 双击 | 点击 `![](path)` |
| 视频 | 双击 | — |
| 音频 | 双击 | — |
| 数学公式 | 点击 | 直接编辑 |
| 脚注 | 悬停 | 直接编辑 |
| Wiki 链接 | 悬停 | 点击 |
| 表格 | 工具栏 | 右键菜单 |
| 拼写检查 | 右键 | 右键 |

## 弹窗导航技巧

### 焦点流程
1. 弹窗打开时第一个输入框获得焦点
2. `Tab` 向前移动到各字段和按钮
3. `Shift + Tab` 向后移动
4. 焦点在弹窗内循环

### 快速编辑
- 简单的 URL 修改：编辑后按 `Enter`
- 取消：从任意字段按 `Escape`
- 多行内容（脚注、数学公式）：使用 `Mod + Enter` 保存

### 鼠标行为
- 点击弹窗外部关闭（更改将被丢弃）
- 悬停弹窗（链接、脚注、Wiki 链接）显示前有 300ms 延迟
- 将鼠标移回弹窗可保持其打开状态
