# VMark 入门指南

VMark 是一款以本地为先的 Markdown 编辑器，具备双模式编辑、丰富的格式工具以及出色的中日韩（CJK）文本支持。

## 快速开始

1. 从[下载页面](/zh-CN/download)**下载并安装** VMark
2. **启动应用**，立即开始写作
3. 使用 `Cmd/Ctrl + O` 或拖放 `.md` 文件来**打开文件**
4. 使用 `Cmd/Ctrl + Shift + O` **打开文件夹**，进入工作区模式

## 界面概览

### 主要区域

- **编辑器**：撰写文档的主要写作区域
- **侧边栏**：文件树导航（使用 `Ctrl + Shift + 2` 切换）
- **大纲**：文档结构视图（使用 `Ctrl + Shift + 1` 切换）
- **状态栏**：字数统计、字符统计和自动保存状态（使用 `F7` 切换）
- **终端**：集成的 Shell 面板（使用 `` Ctrl + ` `` 切换）

### 菜单栏

- **文件**：新建、打开、保存、导出操作
- **编辑**：撤销/重做、剪贴板、查找/替换、文档历史
- **块**：标题、列表、引用块、行操作
- **格式**：文本样式、链接、文本转换
- **视图**：编辑模式、侧边栏、专注/打字机模式
- **工具**：文本清理、中日韩格式化、图片管理

### 编辑模式

VMark 支持两种编辑模式，可随时切换：

| 模式 | 描述 | 快捷键 |
|------|------|--------|
| 富文本 | 实时格式化的所见即所得编辑 | 默认 |
| 源码 | 带语法高亮的原始 Markdown | `F6` |

### 视图模式

通过以下视图模式增强写作专注度：

| 模式 | 描述 | 快捷键 |
|------|------|--------|
| 专注 | 高亮当前段落 | `F8` |
| 打字机 | 保持光标居中 | `F9` |
| 自动换行 | 切换折行显示 | `Alt + Z` |

## 基本格式

### 文本样式

| 样式 | 语法 | 快捷键 |
|------|------|--------|
| **粗体** | `**text**` | `Cmd/Ctrl + B` |
| *斜体* | `*text*` | `Cmd/Ctrl + I` |
| ~~删除线~~ | `~~text~~` | `Cmd/Ctrl + Shift + X` |
| `代码` | `` `code` `` | `Cmd/Ctrl + Shift + `` ` `` |

### 块级元素

- **标题**：使用 `#` 符号或 `Cmd/Ctrl + 1-6`
- **列表**：以 `-`、`*`、`1.` 或 `- [ ]`（任务列表）开头
- **引用块**：以 `>` 开头，或使用 `Alt/Option + Cmd + Q`
- **代码块**：使用三个反引号，可选指定语言
- **表格**：使用"格式"菜单或 `Cmd/Ctrl + Shift + T`

## 文件操作

### 创建和打开

- **新建文件**：`Cmd/Ctrl + N`
- **打开文件**：`Cmd/Ctrl + O`
- **打开文件夹**：`Cmd/Ctrl + Shift + O`（工作区模式）

### 保存

- **保存**：`Cmd/Ctrl + S`
- **另存为**：`Cmd/Ctrl + Shift + S`
- **自动保存**：默认启用，可在设置中配置

### 导出

- **导出 HTML**：使用**文件 → 导出 HTML** —— 包含交互式 VMark 阅读器
- **导出 PDF**：使用打印（`Cmd/Ctrl + P`）并另存为 PDF
- **复制为 HTML**：`Cmd/Ctrl + Shift + C`

导出的 HTML 包含 VMark 阅读器，提供目录、设置面板等功能。[了解更多 →](/zh-CN/guide/export)

## 设置

使用 `Cmd/Ctrl + ,` 打开设置，可自定义：

- **外观**：主题、字体、字号、行高
- **编辑器**：自动保存间隔、默认行为
- **文件与图片**：资源管理、文档工具
- **集成**：AI 提供商、MCP 服务器
- **语言**：中日韩排版规则
- **Markdown**：导出选项、格式偏好
- **快捷键**：自定义键盘快捷键
- **终端**：终端字号和行高

## AI 写作辅助

VMark 内置 AI 精灵 —— 选中文字并按 `Mod + Y`，即可使用 AI 润色、扩展、翻译或转换你的文字。在**设置 > 集成**中配置你偏好的提供商。

[了解更多 AI 精灵 →](/zh-CN/guide/ai-genies) | [配置提供商 →](/zh-CN/guide/ai-providers)

## 入门技巧

1. **使用大纲导航**：点击大纲条目跳转到对应章节
2. **试用 AI 精灵**：选中文字并按 `Mod + Y` 进行润色、改写或扩展
3. **尝试专注模式**：`F8` 帮助你集中注意力于当前段落
4. **学习快捷键**：查看[快捷键指南](/zh-CN/guide/shortcuts)提升效率
